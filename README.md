# ポートフォリオ | Urakawa Haruto

大学での経験と、独学で制作した「日記内省アプリ」と「自己紹介Webサイト」をまとめています。

## 🌱 自己紹介Webサイト
大学での学び・経験・スキルを整理し、読みやすさと構成にこだわったWebページです。  
→ `自己紹介` ボタンから閲覧できます。

## 📝 日記アプリ
日々の出来事を記録し、要約・感情タグ・設問によって内省を促します。  
「行動の理由を理解する」ためのきっかけになることを目指しています。

### 主要機能
- localStorage によるデータ保存（閉じても消えない）
- 感情タグの自動分類
- 要約生成
- 質問と自分の回答の記録
- 検索 / タグ絞り込み
- 編集 / 削除
- ライト / ダークテーマ切り替え

### 使用技術
- **HTML / CSS / JavaScript**
- フレームワーク不使用（UI・動作を自力で実装）

## 🎯 制作の背景
大学の研究で「偶然に見える出来事は、実は無意識の関心が導いた潜在的選択かもしれない」という考えに触れ、  
自分の行動を客観的に振り返るための手段として日記アプリを制作しました。
